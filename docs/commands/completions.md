# bivvy completions

Generates shell completion scripts for tab-completion support.

## Usage

```bash
bivvy completions bash
```

```bash
bivvy completions zsh
```

```bash
bivvy completions fish
```

```bash
bivvy completions powershell
```

```bash
bivvy completions elvish
```

## Supported Shells

| Shell | Value |
|-------|-------|
| Bash | `bash` |
| Zsh | `zsh` |
| Fish | `fish` |
| PowerShell | `powershell` |
| Elvish | `elvish` |

See the [Shell Completions Guide](../guides/shell-completions.md) for
